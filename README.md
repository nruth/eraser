Running the demo
================

0. (sudo) gem install bundler
1. (sudo) bundle install
2. gcc -o xor xor.c
3. copy a binary file to media/test.mp3
4. ./demo.rb or explore for yourself

Running tests
=============

1. bundle install --with=development
2. run `rspec spec`

DISCLAIMER
==========

This probably doesn't run on Windows.
